Polish your Images on a Website!




When you first insert an image into a HTML document, it looks pretty basic. The borders are harsh, and there's not much definition.

You can add some extra CSS styles to make them that bit more special.


To add CSS styles to your HTML, first declare a class for your element, it can be called anything.

<img src="img1.jpg" alt="mountains" class="myclass">

Then, in your CSS document, add an entry for your new class.

.myclass {}

Then, you can add each property:

.myclass {border-style: ridge; border-color: chartreuse; border-width: 10px;}




This image has a border on it, you can do this using the CSS border properties, they are as follows:

border-color, border-style & border-width

border-style is a required property, you must use it for the border to work. It defines what the border will look like, you can use solid, dotted, dashed, double, groove, ridge, inset or outset. Click on the buttons to see what each look like.

border-color defines the colour of the border. You can use colour names, hex colour codes and rgb/rgba colour codes.

border-width defines how thick the border is, you can use any unit supported by html, but pixels (px) is the most common.

If you are competent with using the border properties, you may be interested in learning to use the shorthand border property. It allows you to compact all this code into one declaration:

border: 10px ridge chartreuse;

First specify the width, then the style, and finally the colour. Separate them with spaces. This is useful to keep your CSS more organised and save time typing.



You can take your borders one step further. It is possible in CSS to make every border different.

You can use the CSS properties border-top-style, border-top-width, border-top-color etc, but this means using 12 properties to style the whole image (3 for each side). Instead, we can use the shorthand property we just learnt to declare the value of all three properties at once.

.myclass {border-top: 10px solid green, border-right: 10px dashed yellow; border-bottom: 10px double gray; border-left: 10px ridge red;}

Be careful with this though, because using too much variety can make your page look too busy and make it harsh to view.



This image has a border radius. See that subtle curve on the corners? That's what a radius does. It adds a little more detail to the image and can make it look more professional.

Access this by using the CSS property border-radius and setting it to the appropriate value. Use pixels (px) or a percent value e.g. 30%.

You can also style each corner individually using border-top-left-radius, border-top-right-radius, border-bottom-right-radius & border-bottom-left-radius



If your image is a part of a header or not a main focus of the page, you should consider disabling dragging. Try dragging this image and then try dragging one of the previous ones. Do you see the difference?

This effect can be achieved by using the draggable attribute on the image.

<img src="img1.jpg" alt="mounatin" draggable="false">
