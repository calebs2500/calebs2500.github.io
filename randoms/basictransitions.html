Make a div that changes its properties somehow:

div#testOne { width: 100px; height: 100px; background: red; } div#testOne:hover { background: purple; }


To animate the change, add a transition property to the default state

transition: background 1.0s ease;

Specify the property to animate, the time for the animation to take place over and the type of animation

div#testTwo { width: 100px; height: 100px; background: red; transition: background 1s ease; } div#testTwo:hover { background: purple; }
